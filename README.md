# Mentis – Mental Health & Psychology Management Website

## Overview

Mentis is a web-based platform focused on psychology and mental health development. It was developed as part of the PIDEV – 3rd Year Engineering Program at Esprit School of Engineering – Tunisia during the Academic Year 2025–2026.

The platform provides tools to manage psychological sessions, mental health activities, and user engagement. It allows administrators and users to organize sessions, participate in quizzes, track mood patterns, and access helpful content related to mental well-being.

## Features

- Mental health quiz system  
- Mood tracking and monitoring  
- Session management  
- Event booking system  
- User management (authentication & roles)  
- Content management for psychology resources  
- Responsive and interactive web interface  

## Tech Stack

### Frontend

- Twig (Symfony templating engine)  
- HTML5 / CSS3 / JavaScript  
- Bootstrap (optional, if you used it)  

### Backend

- Symfony 6.4 (PHP Framework)  

### Database

- MySQL  

### Development Tools

- Visual Studio Code / PhpStorm  
- Composer  
- Symfony CLI  

## Architecture

Mentis follows the MVC (Model-View-Controller) architecture provided by Symfony.

- Model: Handles database interactions using Doctrine ORM  
- View: Built with Twig templates for rendering UI  
- Controller: Manages application logic and user requests  

The Symfony backend processes user interactions and communicates with the MySQL database to store and retrieve data such as sessions, quizzes, events, and mood tracking information.

# Mentis – Mental Health & Psychology Management Website

## Overview
Mentis is a desktop application/Website focused on psychology and mental health development. It was developed as part of the PIDEV – 3rd Year Engineering Program at Esprit School of Engineering – Tunisia during the Academic Year 2025–2026.

The application provides tools to help manage psychological sessions, mental health activities, and user engagement. It allows administrators and users to organize sessions, participate in quizzes, track mood patterns, and access helpful content related to mental well-being.

## Features
- Mental health quiz system
- Mood tracking and monitoring
- Session management
- Event booking system
- User management
- Content management for psychology resources
- Interactive graphical interface

## Tech Stack

### Frontend
- JavaFX

### Backend
- Java

### Database
- MySQL

### Development Tools
- IntelliJ IDEA

## Architecture
The Mentis application follows a modular architecture that separates the graphical interface, business logic, and database layer.

The JavaFX interface interacts with the Java backend, which processes user requests and communicates with the MySQL database to store and retrieve information such as sessions, quizzes, events, and mood tracking data.

## Getting Started

### Prerequisites
Make sure you have the following installed:
- Java JDK
- MySQL Server
- JavaFX SDK
- IntelliJ IDEA
- symfony 6.4

### Installation
1. Clone the repository from GitHub
2. Open the project in IntelliJ IDEA
3. Configure the MySQL database connection
4. Run the application from the IDE
